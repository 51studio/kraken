# Kraken

Web standards based rendering engine.

## Usage

- Install Desktop Cli globally in development
  - npm i

- Build
  - npm run build:macos
  - npm run build:android
  - npm run build:linux

- Test (Unit Test and Integration Test)
  - npm test
