## Kraken CLI for Desktop

Kraken client support for macOS, Linux. This is a npm module.

```sh
Usage: kraken <filename|url> [-b path] [-u url]

Start a kraken app. Usage:

Options:
  -V, --version                    output the version number
  -b --bundle <filename>           Bundle path. One of bundle or url is needed, if both determined, bundle path will be
                                   used.
  -u --url <url>                   Bundle URL. One of bundle or URL is needed, if both determined, bundle path will be
                                   used.
  -m --runtime-mode <runtimeMode>  Runtime mode, debug | release. (default: "release")
  -h, --help                       output usage information
```
