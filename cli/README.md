## Kraken CLI

```sh
Usage: kraken [-b bundlePath] [-u bundleURL]

Start a kraken app. Usage:

Options:
  -V, --version                    output the version number
  -b --bundle <bundle>             Bundle path. One of bundle or url is needed, if both determined, bundlePath will be
                                   used.
  -u --url <url>                   Bundle url. One of bundle or url is needed, if both determined, bundlePath will be
                                   used.
  -m --runtime-mode <runtimeMode>  Runtime mode, debug | release. (default: "release")
  -h, --help                       output usage information
```
